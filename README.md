# CodeAlpha Cyber Security Internship Tasks

This repository contains all the tasks completed during the CodeAlpha Cyber Security Internship.

## Tasks

### Task 1: Basic Network Sniffer
A Python-based network packet sniffer that captures and analyzes network traffic.

### Task 2: Secure Coding Review
A secure coding review project to identify vulnerabilities and recommend security improvements.

### Task 3: Network Intrusion Detection System
A Python-based intrusion detection system for monitoring suspicious network activities.
